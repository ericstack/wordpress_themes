# Axis WordPress Theme

## Description

Axis is a custom-made WordPress theme that has been converted from a Bootstrap theme website. It is designed to be modern, responsive, and user-friendly, making it suitable for various types of websites, including blogs and portfolios.

## Features

- **Responsive Design**: Adapts to different screen sizes and devices.
- **Customizable**: Easily modify the theme to fit your needs.
- **Bootstrap Integration**: Utilizes Bootstrap framework for a clean and modern layout.
- **SEO Friendly**: Built with best practices for search engine optimization.

## Installation

1. Download the theme files.
2. Upload the `axis` folder to the `/wp-content/themes/` directory.
3. Activate the theme through the 'Themes' menu in WordPress.

## Usage

After activating the theme, you can customize it through the WordPress Customizer. Modify colors, fonts, and layout options to match your brand.
